# adapter
基于IAdapter接口对android组件listview的baseadapter和Recyclerview的RecycleAdapter的封装和实现。
## 使用
#### 第一步：添加jitpack依赖
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
#### 第二步：添加远程依赖
  implementation 'com.github.lbasis:adapter:Tag'
#### 最新版本：v1.02
implementation 'com.github.lbasis:adapter:1.0.2'
